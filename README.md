## Prérequis
Sur Debian 9:
### GIT
    sudo apt install git
### APACHE
    sudo apt install apache2
### PHP7
    sudo apt install php
## Mise en place
Dans le repertoire `/var/www/html` faire:

    git clone https://github.com/N4n3x/ppe2TableauDeBord.git
